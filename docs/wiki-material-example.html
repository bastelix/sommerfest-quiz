Kalibrierungsgrundlagen

Zuletzt aktualisiert am 24.03.2026

Einleitung

Die Kalibrierung ist ein wesentlicher Bestandteil der Qualitätssicherung. In diesem Artikel erfahren Sie, wie Sie mit MET/CAL präzise Kalibrierungen durchführen.

Voraussetzungen

  • MET/CAL Version 12 oder höher
  • Gültige Lizenz für das FSC-Modul
  • Konfigurierter IEEE-488 Bus
Hinweis

Stellen Sie sicher, dass alle Referenznormale vor Beginn der Kalibrierung akklimatisiert sind (min. 4 Stunden im Kalibrierraum).

Tipp

Verwenden Sie VSET anstelle von MATH für einfache Zuweisungen – das verbessert die Lesbarkeit Ihrer Prozeduren erheblich.

Warnung

Überschreiben Sie niemals die System-Prozeduren ohne vorheriges Backup. Änderungen an SYS_PROC können die gesamte Kalibrierumgebung beeinflussen.

Gefahr

Bei Hochspannungskalibrierungen (> 1kV) müssen die Sicherheitsvorschriften nach DIN EN 50191 eingehalten werden!

Info

Die aktuelle Norm ISO/IEC 17025:2017 fordert eine dokumentierte Messunsicherheitsanalyse für alle Kalibrierungen.

Beispiel

Eine einfache Spannungsmessung mit dem Keithley 2000:

1.001  RSLT  DCV
1.002  VSET  NOMINAL = 10 V
1.003  IEEE  WRITE "CONF:VOLT:DC 10"
1.004  IEEE  READ? RESULT
1.005  MATH  MEM = RESULT

Sie können die Messzeit mit dem Parameter NPLC anpassen. Höhere Werte ergeben genauere, aber langsamere Messungen:

  • NPLC 0.1 – Schnell (4.5 Digit)
  • NPLC 1 – Standard (5.5 Digit)
  • NPLC 10 – Hochpräzise (6.5 Digit)

Code-Beispiele

Hier ein vollständiges Kalibrierungsbeispiel in Python:

import pyvisa

rm = pyvisa.ResourceManager()
dmm = rm.open_resource("GPIB0::22::INSTR")

# Configure for DC voltage measurement
dmm.write("CONF:VOLT:DC 10")
dmm.write("VOLT:DC:NPLC 10")

# Read measurement
result = float(dmm.query("READ?"))
print(f"Measured voltage: {result:.6f} V")

Und dasselbe als MET/CAL FSC-Prozedur:

1.001  TARGET  Keithley 2000 DC Voltage
1.002  RSLT    DCV 10V
1.003  IEEE    WRITE "CONF:VOLT:DC 10"
1.004  IEEE    WRITE "VOLT:DC:NPLC 10"
1.005  IEEE    READ? RESULT
1.006  MATH    MEM = RESULT
1.007  DISP    Measured: [MEM] V
